Go-android-rpc

Write Android applications with GUI support entirely in Go.

Github

What it is

Go-android-rpc is a framework written in Go and Java. It is a bridge between the device running Android and the application, which code written entirely in Go. Using go-android-rpc it is possible to interact with Android UI from Go code.

Framework features:

  • dynamic creation of UI elements
  • interaction with existing UI elements (set text, size, position etc)
  • listening for UI events (button clicks, touch etc)
  • retrieving a list of elements from layouts described as xml
  • subscribing to sensors (accelerometer, light etc)

Inside the application, all the power of Go can be utilized. Application may use channels, goroutines and the rest of Go magic.

What it is not

Go-android-rpc is not Java replacement. Application built with go-android-rpc works as shared library. Java can not be rejected at the time.
